# user-repo

Przykładowy projekt spring-boot na potrzeby zajęć java-dev na UR 2016.

W branchu jpa_homework (i master po ostatnim merge'u) możecie znaleźć przykładowe (nie idealne) rozwiąznanie zadania domowego.
Model zawiera kilka encji powiązanych relacjami, każda jest dostępna jako REST endpoint, i ma zaimplementowane wszystkie metody HTTP.

Projekt zawiera podstawowy widok w HTML z angularjs - źródła możecie znaleźć w src/main/resources/static, a po uruchomieniu aplikacji to podstawowy widok widoczny pod http://localhost:8080/
